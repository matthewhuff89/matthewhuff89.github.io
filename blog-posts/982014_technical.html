I'm gonna start off slow, cause I don't know where you're at, and I don't wanna lose you. Let's start off with what CSS is in the first place. CSS is Cascading Style Sheets. It's a type of file that creates cascading effects on the HTML it's linked to. It overlaps the HTML, like putting one page on top of another. Except the piece of paper you put on top of the bottom one (the HTML) is covered in fancy graphics, and colors, and moves stuff around on the bottom one.

A major part of using these CSS files, is to change the positioning of the elements (objects, paragraphs, pictures, etc) on the HTML/webpage. You can do this by creating a stylesheet (let's call it for the moment stylesheet.css), and opening the file and changing the attribute, "position". So let's say you have a link, or sentence you want people to be able to see anywhere, you would change that "position: fixed"
