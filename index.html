The Rok Programming Language

I'm working through a book called Build Your Own Lisp to fulfill a long-standing curiosity i've had about how languages are born. I'm also really curious about low-level languages like C, which I feel like are the secret language running a lot of our daily lives. I figured this project would be great for getting me in the habit of building side projects, learning more about functional programming languages, building programming languages, and working with C. It's been a blast, i've also added my own touches such as more descriptive variable names, separating out logic into separate libraries, and implemented some types on my own!
