The Rok Programming Language

I'm working through a book called Build Your Own Lisp to fulfill a long-standing curiosity i've had about how languages are born. I'm also really curious about low-level languages like C, which I feel like are the secret language running a lot of our daily lives. I figured this project would be great for 1. Getting me in the habit of building side projects, 2. Teaching more about functional programming languages, 3. Actually building programming languages, and 4. Learning how to build with C.
